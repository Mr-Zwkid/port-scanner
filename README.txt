端口扫描器使用说明
==================

1. 运行环境
-----------
- Python 3.11
- Windows 11

2. 快速开始
-----------
1) 运行 main.py
2) 输入目标IP地址
3) 输入端口号
4) 选择扫描方式
5) 点击对应按钮开始扫描

3. 功能说明
-----------
- ICMP Scan: 检测主机存活
- TCP Connect: 完整TCP连接扫描
- TCP SYN: 半开放扫描
- TCP FIN: FIN标志扫描
- UDP Scan: UDP端口扫描
- Save Log: 保存日志
- Clear Results：清除结果

4. 重点文件说明
- 项目报告：project_report.pdf
- 主函数：main.py
- 源码：src

5. 注意事项
-----------
- 请在授权情况下使用
- 扫描结果保存在output文件夹
- 使用Save Log按钮保存结果
- 具体信息参见 README.md
